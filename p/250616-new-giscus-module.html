上线评论功能～

添加基于 Giscus 的评论系统
Diary
Xuancong Meng 6/16/2025

终于给博客加上了评论功能！🎉

选用了 giscus 作为评论系统，它基于 GitHub Discussions，完全开源且免费。最重要的是，它支持 Markdown 语法，可以让我们能更好地交流想法。

现在，你可以在每篇文章的底部看到评论区了。欢迎来聊聊天呀～
